## Descargar en Github
Te recomendamos descargarlo directamente desde GitHub. Si eres hábil con git, puedes clonarlo en 'Documents\Arduino\libraries' usando el comando.
```bash
Git clone https://github.com/BigtronicaSAS/BigBot
```
